# pigmax-app
Pigmax v5.8 — Ультра-защищенный P2P мессенджер с открытым исходным кодом. Работает без центральных серверов, использует сквозное шифрование и уникальную систему длинных ключей (1200+ символов). Поддерживает кастомизацию интерфейса, биометрическую аутентификацию и полную очистку данных одним нажатием. Конфиденциальность превыше всего.
